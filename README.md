# Story AI

StoryAI 프로젝트는 인공지능을 활용하여 소설을 자동으로 생성하는 프로젝트입니다.
캐릭터를 생성하며 캐릭터들 각각의 시놉시스를 생성할 수 있습니다.

## 요구 사항
- Python 3.11 이상

## 설치 방법
```
pip install https://github.com/paust-team/storyai/releases/download/v0.1.0/storyai-0.1.0-py3-none-any.whl
```

## 사용 방법
### 프로젝트 초기화
```
mkdir ./storyai_example
cd ,/storyai_example
storyai init

openai_api_key: <your openai api key>
```
### 캐릭터 생성
```
storyai persona add

Name: <character's name>
Age: <character's age>
MBTI: <character's MBTI>
Gender: <character's Gender(MALE or FEMALE)>
Description: <character's description(optional)>

Added persona ID: ...
```
### 시놉시스 생성
```
storyai synopsis add

Theme: <theme>
Persona ID: <persona id>
```
### 생성된 시놉시스 조회
```
storyai synopsis show --synopsis-id=<synopsis id>

Synopsis:
  ...
```

## 예시
### 캐릭터 생성
```
storyai persona add --name "박진성" --age 33 --description "" --mbti ENTP --gender MALE


Added persona ID: 2, context: "박진성" 페르소나는 MBTI 유형이 "ENTP"인 33세 남성이다. ENTP는 외향적인 성격으로 알려져 있으며, 혁신적이고 창의적인 아이디어를 가지고 있으며 논쟁적이고 호기심이 많다. 따라서 "박진성" 페르소나에게는 아래와 같은 캐릭터 특징이 어울릴 것이다:

1. "박진성"은 혁신적인 아이디어와 창의적인 해결책을 제시하는 능력을 가지고 있다. 이러한 특징을 가진 캐릭터로는 "토니 스타크 (아이언맨)"이 좋은 예시일 수 있다. 토니 스타크는 엔지니어이자 천재적인 아이디어로 많은 혁신적인 발명품을 개발한다.
2. ENTP는 논쟁과 토론을 즐기는 성격이다. 이를 반영한 캐릭터로는 "셜록 홈즈"가 좋은 예시일 수 있다. 셜록 홈즈는 논리적으로 사고하는 형사로써, 사건을 해결하기 위해 상황을 분석하고 명확한 이유에 기반한 논리적 주장을 제시한다.
3. "박진성"은 호기심이 많은 성격으로서, 새로운 경험과 도전을 즐긴다. 이와 비슷한 특징을 가진 캐릭터로는 "잭 스패로우 (파이레이츠 오브 카리비안)"이 소개될 수 있다. 잭 스패로우는 모험을 즐겨 하며 새로운 땅이나 보물을 찾기 위해 여행을 떠나는 등 일상에 지루함을 느끼지 않는다.
4. ENTP는 사회적으로 재능이 있으며 어떤 모임에서나 리더십을 발휘하기도 한다. 이런 특징을 갖춘 캐릭터로는 "프란시스 언더우드 (하우스 오브 카드)"가 소개될 수 있다. 프란시스 언더우드는 정치인으로서 정치권에서 권력을 얻는 방법을 잘 알고 있으며, 냉철한 판단력과 전략적인 사고로 일을 추진시킨다.
5. 마지막으로, ENTP는 타인과의 관계에 있어서 적극적이며 카리스마도 지니고 있다. 이런 특징을 가진 캐릭터로는 "토니 스타크 (아이언맨)"와 "토르 (마블 시네마틱 유니버스)" 등이 소개될 수 있다. 토니 스타크는 자신의 아이디어와 개성으로 동료들에게 환호받는 인물이며, 토르는 용감하고 카리스마 있는 존재로서 동
```

### 시놉시스 생성
```
storyai synopsis add 
Theme: 박진성이 스타트업을 창업하고 어려움을 겪지만 극복하는 스토리.
Character ID: 2

Added synopsis ID: 2, content: - 제목: "혁신의 길, 박진성의 스타트업"...
```

### 생성된 시놉시스 조회
```
storyai synopsis show --synopsis-id 2

Synopsis: 
- 제목: "혁신의 길, 박진성의 스타트업"

시놉시스:
"혁신의 길, 박진성의 스타트업"은 혁신적인 아이디어와 창의적인 해결책을 가지고 있는 박진성이 스타트업을 창업하고 어려움을 겪으며 극복하는 이야기를 그린 영화이다.

박진성은 외향적인 성격과 혁신적인 아이디어를 가진 ENTP 유형의 천재로, 자신의 열정과 창의력을 삶의 일부로 삼고 싶어하는 꿈을 품고 있다. 그의 첫 번째 스타트업 아이디어는 혁신적인 기술을 활용한 에너지 저장장치인 "스마트 에너지박스"다. 하지만 경쟁력 있는 시장과 자금조달의 어려움으로 인해 박진성의 스타트업은 큰 위기에 놓인다.

그러나 그는 절대 포기하지 않으며 다양한 도전과 실패를 경험하며 성장한다. 그의 창의적인 해결책과 뛰어난 커뮤니케이션 능력은 투자자들의 관심을 끌어내고, 온라인으로 큰 주목을 받게 된다. 박진성과 그의 팀은 힘겨운 여정을 통해 창업의 어려움과 위기를 극복하고, 성공을 찾아나간다.

"혁신의 길, 박진성의 스타트업"은 성공에 도달하기까지의 노력과 역경을 이기는 힘으로 가득한 박진성의 이야기를 통해 스타트업 창업자의 고난과 열정을 그린 영화이다. 이 영화는 혁신적인 아이디어와 희생을 통해 얻어낸 성공의 의미를 전하며, 꿈과 열정을 향한 여정에서 우리에게 감동과 영감을 주는 작품이다.
```
